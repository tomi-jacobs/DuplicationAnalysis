Gene duplication events across 39 Aizoaceae taxa were mapped onto a coalescent species tree using PhyParts, drawing from 13,822 duplicate subtrees extracted from 15,818 homolog gene trees. Duplicated genes were functionally annotated using PANTHER 19.0, yielding 1,636 total annotated entries across 11 nodes. Duplications were concentrated in the rapidly radiating Ruschioideae clade, with Node 5 showing the highest count at 602 unique genes. Key duplicated gene families included phosphoenolpyruvate carboxylase, glycosyltransferases, ABC transporters, and heat shock proteins, suggesting that gene family expansion in Ruschioideae was enriched for genes tied to CAM/C4 photosynthesis, stress response, and secondary metabolism.
